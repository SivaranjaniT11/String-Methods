String Methods

1 concat() joins two or more strings

ex:
a="Beautiful"
b="flowers"

2 length() - length of a string

ex:
a="Image"

3 Extracting String Parts

(1)slice(start, end)

ex:
a="Imagepic"

(2)substring(start, end)

ex:
a="Imagepic"

(3)subst(start, length)

ex:
a="Imagepic"

4 String toUpperCase() and String toLowerCase()

convert upper case and lower case

ex. a="Image"

5 String charAt() and String charCodeAt()

charAt() method returns the character at a specified index (position) in a string

ex. a="Image"

charCodeAt() method returns the unicode of the character at a specified index in a string

ex. a="Image"

6 String replace() and String replaceAll()

ex. It's My home

ex. I love cats. i like cats

7 String trim()

Trim() method removes whitespace from both sides of a string

ex. welcome my home!

string trimstart() and string trimend()

trimstart-Remove the starting whitespace and trimend()-Remove the Ending whitespace

8 String Padding

padStart() and padEnd() to support padding at the beginning and at the end of a string.

padStart() ex:5

padEnd() ex:5

String split()

A string can be converted to an array with the split() method

ex. a=a,b,c,e,f
